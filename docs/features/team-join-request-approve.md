# 팀 가입 신청 승인

## 기능 목적

- 팀장이 가입 신청을 승인할 수 있다.
- 승인된 회원은 해당 팀의 팀원이 된다.

## 비즈니스 규칙

- 로그인한 회원만 요청할 수 있다.
- 존재하는 팀만 승인할 수 있다.
- 해당 팀의 팀장만 가입 신청을 승인할 수 있다.
- 존재하는 가입 신청만 승인할 수 있다.
- 해당 팀에 들어온 가입 신청만 승인할 수 있다.
- 신청 회원이 이미 다른 팀에 속해 있다면 승인할 수 없다.
- 승인된 회원의 TeamRole은 MEMBER이다.
- 승인 완료 후 가입 신청 데이터는 삭제한다.
- TeamMember 생성과 가입 신청 삭제는 하나의 트랜잭션에서 처리한다.

## 처리 순서

1. JWT에서 로그인 회원 ID를 가져온다.
2. Team과 leaderMember를 조회한다.
3. 로그인 회원이 팀장인지 확인한다.
4. 가입 신청과 신청 회원을 조회한다.
5. 가입 신청이 해당 팀의 신청인지 확인한다.
6. 신청 회원이 이미 팀에 속해 있는지 다시 확인한다.
7. TeamMember를 생성한다.
8. 가입 신청을 삭제한다.
9. 승인 결과를 반환한다.

## API

POST /api/teams/{teamId}/join-requests/{requestId}/approve

## 테스트 항목

- 가입 신청 승인 성공
- 승인 후 TeamMember 생성 확인
- 승인 후 가입 신청 삭제 확인
- 팀장이 아닌 회원의 승인 실패
- 존재하지 않는 팀 승인 실패
- 존재하지 않는 가입 신청 승인 실패
- 다른 팀의 가입 신청 승인 실패
- 이미 팀에 속한 회원 승인 실패
