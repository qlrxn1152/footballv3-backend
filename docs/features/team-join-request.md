# 팀 가입 신청

## 기능 목적

- 팀에 소속되지 않은 회원이 원하는 팀에 가입 신청을 보낼 수 있다.

## 비즈니스 규칙

- 로그인한 회원만 가입 신청을 할 수 있다.
- 존재하는 팀에만 가입 신청을 할 수 있다.
- 이미 팀에 가입한 회원은 가입 신청을 할 수 없다.
- 한 회원은 동시에 하나의 가입 신청만 가질 수 있다. ( 기능이 어느정도 완성되고, 여러개의 가입신청을 넣을수있도록 변경예정 )
- 가입 신청 시각을 저장한다.
- 가입 신청이 생성됐다고 해서 바로 TeamMember가 생성되지는 않는다.
- 가입신청이 수락되면, 우선 테이블 기록을 없앤다. ( 기능이 어느정도 완성되고, 이후에는 기록을 남기는식으로 변경)

## 처리 순서

1. JWT에서 로그인 회원 ID를 가져온다.
2. teamId로 팀을 조회한다.
3. memberId로 회원을 조회한다.
4. 회원이 이미 팀에 가입했는지 확인한다.
5. 기존 가입 신청이 있는지 확인한다.
6. TeamJoinRequest를 저장한다.
7. 가입 신청 결과를 반환한다.

## API

POST /api/teams/{teamId}/join-requests

## 테스트 항목

- 팀 가입 신청 성공
- 존재하지 않는 팀 신청 실패
- 존재하지 않는 회원 신청 실패
- 이미 팀에 가입한 회원 신청 실패
- 중복 가입 신청 실패


## 구현 결과

- JWT 회원 ID를 이용한 가입 신청
- 존재하는 팀에만 신청 가능
- 이미 팀에 가입한 회원 신청 방지
- 동일 회원의 중복 신청 방지
- 가입 신청 시각 저장
- TeamMember는 승인 전까지 생성하지 않음

## 테스트 결과

- 팀 가입 신청 성공
- 존재하지 않는 팀 신청 실패
- 팀 소속 회원 신청 실패
- 중복 가입 신청 실패
- 전체 테스트 통과
