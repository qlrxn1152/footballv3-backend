# 팀 상세 조회

## 기능 목적

- 사용자가 특정 팀의 상세 정보를 조회할 수 있다.

## 조회 정보

- 팀 ID
- 팀 이름
- 팀 레이팅
- 팀장 ID
- 팀장 아이디
- 팀 생성 일시
- 현재 팀원 수
- 소속 팀원 목록

## 비즈니스 규칙

- 존재하는 팀만 조회할 수 있다.
- 팀 상세 조회는 로그인하지 않은 사용자도 가능하다.
- 팀원 목록에는 회원 ID, 아이디, 레이팅, 팀 역할, 가입 일시를 표시한다.
- 팀 생성자는 LEADER 역할로 조회된다.
- 팀원은 가입 일시가 빠른 순서로 조회한다.
- 팀원 수는 조회된 TeamMember의 개수로 계산한다.

## 처리 순서

1. teamId로 Team과 팀장 Member를 조회한다.
2. 팀이 존재하지 않으면 예외를 발생시킨다.
3. 해당 팀의 TeamMember와 Member를 조회한다.
4. TeamMember를 팀원 응답 DTO로 변환한다.
5. 팀 정보와 팀원 목록을 상세 응답 DTO로 반환한다.

## API

GET /api/teams/{teamId}

- Team과 leaderMember Fetch Join 조회
- TeamMember와 Member Fetch Join 조회
