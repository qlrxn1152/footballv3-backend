# 팀 가입 신청 목록 조회

## 기능 목적

- 팀장이 자신의 팀에 들어온 가입 신청 목록을 조회할 수 있다.

## 비즈니스 규칙

- 로그인한 회원만 요청할 수 있다. ( 팀장 )
- 존재하는 팀만 조회할 수 있다.
- 해당 팀의 팀장만 가입 신청 목록을 조회할 수 있다.
- 가입 신청 회원의 ID, 아이디, 레이팅, 신청 시각을 반환한다.
- 가입 신청이 오래된 순서대로 조회한다.
- 가입 신청이 없다면 빈 목록을 반환한다.

## 처리 순서

1. JWT에서 로그인 회원 ID를 가져온다.
2. teamId로 Team과 leaderMember를 조회한다.
3. 요청한 회원이 해당 팀의 팀장인지 확인한다.
4. 해당 팀의 TeamJoinRequest와 Member를 조회한다.
5. 가입 신청 응답 DTO로 변환한다.
6. 가입 신청 목록을 반환한다.

## API

GET /api/teams/{teamId}/join-requests

## 테스트 항목

- 팀장의 가입 신청 목록 조회 성공
- 가입 신청이 없는 경우 빈 목록 반환
- 일반 회원의 조회 실패
- 존재하지 않는 팀 조회 실패
- 가입 신청 시간순 정렬
