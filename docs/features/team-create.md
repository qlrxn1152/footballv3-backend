# 팀 생성

## 기능 목적

- 로그인한 회원이 새로운 팀을 생성할 수 있다.

## 요청 정보

- 팀 이름 ( String )

## 생성되는 정보

- 팀 ID
- 팀 이름
- 팀장
- 생성 일시
- 팀 레이팅

## 기본값

- 팀 레이팅 : 1500
- 팀장 : 팀을 생성한 회원

## 비즈니스 규칙

- 로그인한 회원만 팀을 생성할 수 있다.
- 팀 이름은 비어 있을 수 없다.
- 팀 이름은 2자 이상 10자 이하로 제한한다.
- 팀 이름은 중복될 수 없다.
- 팀 이름의 앞뒤 공백은 제거한다.
- 팀 생성자는 자동으로 팀장이 된다.
- 팀 생성자는 자동으로 해당 팀의 멤버가 된다.
- 팀 생성과 팀 멤버 등록은 하나의 트랜잭션으로 처리한다.
- 1명의 멤버는 여러개의 팀이아닌 1개의 팀에만 속할수있다.

## 처리 순서

1. JWT에서 로그인 회원 ID를 가져온다. 
2. 팀 이름을 정규화한다. ( 공백제거 .. )
3. 팀 이름 중복 여부를 확인한다.
4. 회원을 조회한다.
5. Team을 저장한다.
6. 생성자를 TeamMember로 저장한다. ( TeamMember -> 팀과, 속해있는 멤버들을 확인할 수 있는)
7. 생성된 팀 정보를 반환한다. ( dto )
