# 팀 목록 조회

## 기능 목적

- 사용자( 로그인회원 / 비회원 모두 가능)가 생성된 팀 목록을 조회할 수 있다.

## 조회 정보

- 팀 ID
- 팀 이름
- 팀장 아이디
- 팀 레이팅
- 팀 생성 일시

## 비즈니스 규칙

- 팀 목록은 생성 일시가 최신인 순서로 조회한다.
- 등록된 팀이 없으면 빈 배열을 반환한다.
- 팀 목록 조회는 로그인하지 않은 사용자도 가능하다.

## 처리 순서

1. Team 데이터를 생성일 최신순으로 조회한다. ( 내림차순 정렬 )
2. 각 Team을 TeamListResponse로 변환한다. 
3. 팀 목록을 반환한다. ( List< TeamListResponse > )

## API

GET /api/teams

## 테스트 항목

- 팀 목록 조회 성공
- 최신 팀이 먼저 조회됨
- 팀이 없으면 빈 배열 반환


## 구현 결과
- 생성된 팀 전체 조회
- 최신 생성 순서로 정렬
- 로그인 없이 조회 가능
- 팀이 없으면 빈 배열 반환

## 테스트 결과

- 팀 목록 조회 성공
- 빈 목록 조회 성공
- 전체 테스트 통과
